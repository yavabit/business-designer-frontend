# Конструктор бизнес процессов

Приложение, позволяющее пользователям создавать свои бизнес-процессы и бизнес-схемы с использованием drag&drop подхода

## Стек

- React 19
- TS (SWC)
- Vite
- AntD
- Redux Toolkit
- RTK Query
- React Flow
- React icons
